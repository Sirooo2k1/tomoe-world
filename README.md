# 共笑®︎ Official Site

Website được xây dựng bằng React.js và Tailwind CSS.

## Cài đặt

```bash
npm install
```

## Chạy development server

```bash
npm run dev
```

## Build cho production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Cấu trúc dự án

```
tomoe_world/
├── src/
│   ├── components/
│   │   ├── Header.jsx      # Header với navigation
│   │   ├── Hero.jsx        # Hero section với background image
│   │   ├── Content.jsx     # Nội dung chính
│   │   ├── InstagramFeed.jsx # Instagram feed
│   │   └── Footer.jsx      # Footer
│   ├── App.jsx             # Component chính
│   ├── main.jsx            # Entry point
│   └── index.css           # Tailwind CSS
├── index.html
├── package.json
├── vite.config.js
└── tailwind.config.js
```
